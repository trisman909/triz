# Lanternfall 1.0.0-rc.1

## Technology candidate scope

- Hub-first deterministic action-roguelite loop
- Five Bearer class definitions, five biome themes, 40 enemy definitions and
  15 multi-phase guardian definitions
- Echo resonance builds, run economy, quests and atomic profile saves
- Keyboard/mouse and controller input with persisted binding overrides
- Scalable HUD, reduced motion, subtitles, high-contrast preferences and
  configurable presentation intensity
- Original adaptive procedural score
- 100 progression, exploration and mastery achievements

This build proves the production packaging, manifest, checksums and executable
startup pipeline. It is not yet a content-complete commercial candidate.
Runtime/content gaps are tracked in `Docs/COMPLETION_AUDIT.md` and Phases 13–17
of `Docs/PROJECT_PLAN.md`.
